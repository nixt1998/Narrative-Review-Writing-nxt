# Final Acceptance Checklist

Run this checklist at Step 10.0 before delivering the final manuscript. Every item must pass.

## Anti-Hallucination
- [ ] All citations from user-provided, verified literature
- [ ] Every PMID/DOI verified against actual PDF content
- [ ] Zero fabricated references
- [ ] Zero search-engine-only citations

## Structure
- [ ] All framework headings are conclusion statements (not descriptive labels)
- [ ] Logical chain continuous from Introduction through Conclusion
- [ ] Figure/table positions marked in text

## Content Quality
- [ ] "Laundry list" style absent -- every section synthesizes across studies
- [ ] Critical evaluation present for key cited studies
- [ ] Balanced coverage -- conflicting evidence addressed, not ignored
- [ ] Evidence-based reasoning throughout
- [ ] No overconfident or dogmatic language
- [ ] Original research cited (not reviews of reviews)

## Conclusion
- [ ] 3-5 sentence synthesis (not a summary list)
- [ ] Limitations of this review stated
- [ ] Resolved vs unresolved questions identified
- [ ] Future directions with solution pathways
- [ ] Length: 350-500 words

## Figures and Tables
- [ ] Figure count matches Step 1.0 parameter
- [ ] Each figure legend is a core conclusion (not description)
- [ ] Abbreviations complete per figure
- [ ] Color guidance provided (color-blind safe + grayscale compatible)
- [ ] Table count matches Step 1.0 parameter
- [ ] Tables are synthesis-level with traceable citations

## References
- [ ] 100% citation traceability (every [N] has a reference)
- [ ] Every reference cited at least once in text
- [ ] Format: Vancouver style
- [ ] Citation format: [N] before period

## Final Assembly
- [ ] Abbreviations table complete and accurate
- [ ] Cross-section consistency check passed (Step 4.5)
- [ ] Word count within user-specified limit
- [ ] All user confirmation gates documented in self-check log
