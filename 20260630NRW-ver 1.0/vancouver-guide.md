# Vancouver Reference Format Guide

## In-Text Citation

- Numeric, sequential in order of first appearance
- Bracketed before the period: `...as demonstrated [1,3,5].`
- Multiple references: comma-separated `[1,3,7]`
- Consecutive references: hyphenated `[1-4]`

## Reference List Entry Formats

### Journal Article (most common)
```
Author AA, Author BB, Author CC. Title of article. Journal Abbreviation. Year;Volume(Issue):Pages.
```
Example: `Dhillon P. How to write a good scientific review article. FEBS J. 2022;289(13):3592-3602.`

### Journal Article with DOI
```
Author AA. Title. Journal Abbreviation. Year;Volume(Issue):Pages. DOI: 10.xxxx/xxxx.
```

### More than 6 Authors
List first 6 authors then "et al."
```
Page MJ, Moher D, Bossuyt PM, Boutron I, Hoffmann TC, Mulrow CD, et al.
```

### Book Chapter
```
Author AA. Chapter title. In: Editor AA, editor. Book Title. Edition. Place: Publisher; Year. p. xx-xx.
```

## Formatting Rules

- Author names: Last name FirstInitialMiddleInitial. No periods between initials, no spaces
- Article title: Sentence case (only first word and proper nouns capitalized)
- Journal name: NLM-abbreviated (use PubMed abbreviations)
- Year: Followed by semicolon, no space before volume
- Volume(Issue): Issue in parentheses, no space after volume
- Pages: Hyphen between start-end, no "p." prefix
- DOI: "DOI: " prefix, full URL-format DOI

## Verification

Cross-check all references against:
1. PubMed for correct journal abbreviation and PMID
2. The original PDF (Step 3 verification)
3. In-text citation numbering (Step 9 cross-verification)
