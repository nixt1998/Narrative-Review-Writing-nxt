## Quick Reference (Most Common Cases)

# Vancouver Reference Format Guide

## In-Text Citation

- Numeric, sequential in order of first appearance
- Bracketed before the period: `...as demonstrated [1,3,5].`
- Multiple references: comma-separated `[1,3,7]`
- Consecutive references: hyphenated `[1-4]`

## Reference List Entry Formats

### Journal Article (most common)
```
Author AA, Author BB, Author CC. Title of article. Journal Abbreviation. Year;Volume(Issue):Pages.
```
Example: `Dhillon P. How to write a good scientific review article. FEBS J. 2022;289(13):3592-3602.`

### Journal Article with DOI
```
Author AA. Title. Journal Abbreviation. Year;Volume(Issue):Pages. DOI: 10.xxxx/xxxx.
```

### More than 6 Authors
List first 6 authors then "et al."
```
Page MJ, Moher D, Bossuyt PM, Boutron I, Hoffmann TC, Mulrow CD, et al.
```

### Book Chapter
```
Author AA. Chapter title. In: Editor AA, editor. Book Title. Edition. Place: Publisher; Year. p. xx-xx.
```

## Formatting Rules

- Author names: Last name FirstInitialMiddleInitial. No periods between initials, no spaces
- Article title: Sentence case (only first word and proper nouns capitalized)
- Journal name: NLM-abbreviated (use PubMed abbreviations)
- Year: Followed by semicolon, no space before volume
- Volume(Issue): Issue in parentheses, no space after volume
- Pages: Hyphen between start-end, no "p." prefix
- DOI: "DOI: " prefix, full URL-format DOI

## Verification

Cross-check all references against:
1. PubMed for correct journal abbreviation and PMID
2. The original PDF (Step 3 verification)
3. In-text citation numbering (Step 9 cross-verification)

PART 0 -- GENERAL PRINCIPLES
================================================================================

THREE IN-TEXT CITATION SYSTEMS:
  1. Citation-sequence  : numbered in order of appearance in text
  2. Citation-name      : numbered alphabetically by author surname
  3. Name-year          : (Smith 2006) in text; list sorted by author+year

CORE RULES:
  - Cite only sources you have personally seen.
  - Always cite the version you actually used (online != print).
  - Within one reference list, format must be consistent.
  - Priority order for NLM rules:
      1st  MEDLARS indexing manual
      2nd  ANSI/NISO Z39.29-2005
      3rd  ISO 690

PART 1 -- UNIVERSAL AUTHOR / NAME RULES
================================================================================
(Apply to all document types unless otherwise noted)

BASIC FORMAT:
  Surname AB.  (surname first, then up to 2 initials, no spaces between initials)
  Multiple authors: comma + space between names; period at end of author block.

AUTHOR COUNT:
  NLM standard  : list ALL authors regardless of number.
  Acceptable alt: limit to N authors, then add  "et al."  or  "and others".

SPECIAL NAME HANDLING:
  Hyphenated surname       : retain hyphen          Palmer-Canton E
  O', D', L' prefix        : retain                 O'Brien AD
  Other punctuation in name: omit                   St. James -> St James CA
  Surname prefix (van, de) : retain                 van de Kamp JM
  Unhyphenated compound    : retain in full          Lopez Moreno S
  Diacritics / accents     : ignore                 u->u, A->A
  Ligatures                : split                  ae->ae, oe->oe
  Academic degrees         : omit entirely          M.D., Ph.D., F.R.C.S.
  Honorifics               : omit                   Sir, Dr., Colonel
  Generational suffix      : keep after initials,   DeVita VT Jr
                             no punctuation;        Jones JG 2nd
                             Roman->Arabic ordinal   Adams JA 3rd
  Non-Roman scripts        : romanize               (Cyrillic, Arabic, CJK, etc.)

CORPORATE / INSTITUTIONAL AUTHOR:
  - Omit leading "The".
  - List subdivisions high->low, comma-separated:
      American College of Surgeons, Committee on Trauma.
  - Add 2-letter ISO country code if no nationality in name:
      National Cancer Institute (US).
  - Multiple institutions: separate with semicolons.

NO AUTHOR FOUND:
  - If editor/translator exists -> start with editor/translator.
  - If none -> start with title.
  - NEVER use "anonymous" or "anon".

EDITOR FORMAT:
  Last editor name followed by:  , editor.   (singular)
                                 , editors.  (plural)

COMPILER FORMAT:
  Last compiler name followed by:  , compiler.  /  , compilers.

INVENTOR FORMAT:
  Last inventor name followed by:  , inventor.  /  , inventors.
  Then semicolon before assignee field.

PART 2 -- JOURNAL ARTICLES
================================================================================

STANDARD FORMAT:
  Author(s). Article title. Journal Abbr. Year Mon Day;Vol(Issue):Pages.

ELEMENT ORDER (R=Required, O=Optional):
  Author (R) | Author Affiliation (O) | Article Title (R) | Article Type (O) |
  Journal Title (R) | Edition (R) | Type of Medium (R) |
  Date of Publication (R) | Supplement/Part/Special No. (R) |
  Volume (R) | Issue (R) | Location/Pagination (R) |
  Physical Description (O) | Language (R) | Notes (O)

--- ARTICLE TITLE ---
  - Transcribe as found; original language.
  - Capitalize: first word, proper nouns, proper adjectives, acronyms only.
  - Title : Subtitle separated by colon + space (unless title ends in ? ! .)
  - Non-English title -> translate -> place translation in [square brackets].
  - End with period (unless ends in ? or !, or Article Type follows).
  - Article type: [abstract] or [letter] in square brackets after title.
  - Special chars: if Greek letter starts a word, do NOT capitalize it (gamma-linolenic).
    If unavailable, spell out: gamma-linolenic.
    Superscripts/subscripts unavailable -> parentheses: TiO(2).

--- JOURNAL TITLE ---
  - Abbreviate per NLM Catalog / PubMed journal list.
  - One-word journal titles: do NOT abbreviate (e.g., Lancet, Virology).
  - CJK-language journals: do NOT abbreviate.
  - Capitalize all retained words; omit articles, conjunctions, prepositions.
  - Remove ALL punctuation within the abbreviated title.
    Journal of Neuro-Oncology -> J Neurooncol.
  - Omit subtitle: JAMA: the Journal of the American Medical Association -> JAMA.
  - Multiple editions: add edition in parentheses after title abbreviation:
    Am Homeopath (Consum Ed).
  - End with period.
  - Acceptable alternative: use full journal title; or add periods after each
    abbreviation: J. Contin. Educ. Nurs.

--- DATE OF PUBLICATION ---
  Format:  YYYY Mon DD   (e.g., 2005 Jan, 2004 May 5)
  - Month: English 3-letter abbreviation (Jan Feb Mar Apr May Jun
                                           Jul Aug Sep Oct Nov Dec)
  - Translate non-English month names to English.
  - Seasons: translate to English, capitalize, do NOT abbreviate:
    Spring, Summer, Fall, Winter.
  - Roman numeral years -> Arabic.
  - Date ranges:
      Multiple months     : 2005 Jan-Feb
      Cross-year months   : 1999 Dec-2000 Jan
      Multiple years      : 2002-2003  (NOT 2002-03)
      Multiple days       : 2005 Feb 1-7
      Multiple seasons    : Fall-Winter
  - Punctuation after date:
      If volume+issue follow -> semicolon  ;
      If only issue follows  -> colon      :
      If neither follows     -> colon      :

--- SUPPLEMENT / PART / SPECIAL NUMBER ---
  Supplement  -> Suppl   (no period)
  Part        -> Pt      (inside parentheses when attached to volume/issue)
  Special No  -> Spec No
  Arabic numerals only: Suppl II -> Suppl 2
  Pt inside parentheses: 34(Pt 1)
  Named supplement: abbreviate important words: Suppl Dementia
  Translate non-English terms to English equivalent.

--- VOLUME NUMBER ---
  - Omit "volume", "vol.", etc.
  - Arabic numerals only (LX -> 60).
  - Multiple volumes: hyphen: 5-6
  - No punctuation after volume number (unless no issue, then colon).

--- ISSUE NUMBER ---
  - Omit "number", "no.", etc.
  - Arabic numerals only (IV -> 4).
  - Multiple issues: hyphen: 2-3
  - Enclose in parentheses: (1)
  - Follow with colon: (1):
  - Optional: omit if journal has continuous pagination throughout a volume.

--- LOCATION / PAGINATION ---
  - Give full inclusive page range.
  - Do NOT repeat digits unless letter suffix differs:
      123-125  ->  123-5      (but 124A-126A stays as is)
  - Letter prefixes (supplements): S10-8
  - Roman numeral pages: KEEP as Roman (do NOT convert).
  - Discontinuous pages: comma + space between groups: 345-6, 348-9, 352.
    If > 3 groups, use passim for remainder.
  - No page numbers: [5 p.]
  - Unnumbered pages near numbered: [preceding p. 1] or [following p. 503]
  - End with period.

--- LANGUAGE ---
  - Note only if NOT English.
  - Capitalize language name, end with period: French. Spanish. Arabic.
  - Multiple languages: comma-separated, period at end: French, English.

--- NOTES (SELECTED) ---
  Erratum:      Erratum in: [journal citation]
  Retraction:   Retraction in: [journal citation]
  Epub ahead:   Epub 2005 Mar 19.
  DOI:          doi:10.1542/peds.2004-1441.
  (Multiple authors all cited; DOI placed after page number)

STANDARD EXAMPLES:
  Petitti DB, Crooks VC, Buckwalter JG, Chiu V. Blood pressure levels before
  dementia. Arch Neurol. 2005 Jan;62(1):112-6.

  Bhutta ZA, Darmstadt GL, Hasan BS, Haws RA. Community-based interventions for
  improving perinatal and neonatal health outcomes in developing countries: a review
  of the evidence. Pediatrics. 2005 Feb;115(2 Suppl):519-617.
  doi:10.1542/peds.2004-1441.

PART 3 -- ENTIRE BOOKS
================================================================================

STANDARD FORMAT:
  Author(s). Title. Edition. Place: Publisher; Year. Pages.

ELEMENT ORDER (R=Required, O=Optional):
  Author/Editor (R) | Author Affiliation (O) | Title (R) |
  Content Type (O) | Type of Medium (R) | Edition (R) |
  Secondary Authors (O) | Place of Publication (R) |
  Publisher (R) | Date of Publication (R) | Pagination (O) |
  Physical Description (O) | Series (O) | Language (R) | Notes (O)

--- BOOK TITLE ---
  - Original language; capitalize first word + proper nouns/adjectives only.
  - Title : Subtitle -> colon + space.
  - Non-English title -> [English translation] in brackets after original.
  - End with period (unless followed by medium type designation).
  - Special characters: same rules as article title.
  - Multiple equal languages (e.g., Canadian bilingual): join with  =
    Title in English = Titre en francais.

--- EDITION ---
  Required when > 1 edition exists.
  - Arabic ordinal numbers: 2nd ed.  3rd rev. ed.
  - Common abbreviations:
      edition    -> ed.       revised    -> rev.
      enlarged   -> enl.      abridged   -> abr.
      updated    -> upd.      corrected  -> corr.
  - Non-English edition words:
      French   : edition -> ed.
      German   : Auflage -> Aufl.
      Italian  : edizione -> ed.
      Russian  : izdanie -> izd.
      Spanish  : edicion -> ed.
  - First edition: do NOT write "1st ed." unless a later edition is known to exist.
  - "Annual Report", "Final Report", "Interim Report" = edition statements.
  - End with period.

--- SECONDARY AUTHORS (editors, translators, illustrators) ---
  Place after title (and content/medium/edition if present).
  Last name followed by  , editor  /  , editors  /  , translator  /  , illustrators
  Multiple types: semicolon between roles:
    Smith BC, editor; Carson HT, illustrator.
  Same person, multiple roles: Jones AB, editor and translator.

--- PLACE OF PUBLICATION ---
  - City of publication.
  - US/Canada: add 2-letter state/province code in parentheses:
      New York (NY)    Philadelphia (PA)    Ottawa (ON)
  - Other countries: add country name or ISO code in parentheses for
    obscure or ambiguous cities:
      Basel (Switzerland)    Malaga (Spain)    Berne (CH)
  - Use English city name: Vienna (not Wien), Munich (not Munchen).
  - End with colon.
  - Can infer from other sources -> place in [brackets]: [Chicago]:
  - Completely unknown: [place unknown]:

--- PUBLISHER ---
  - Transcribe as found; keep original capitalization and punctuation.
  - Subdivision after main name: McGraw-Hill, Health Professions Division.
  - Joint publication: use first publisher; add second as note if desired.
  - End with semicolon.
  - Unknown: [publisher unknown];

--- DATE OF PUBLICATION ---
  - Year is required; month optional (format: 2004 May).
  - Arabic numerals only (Roman -> Arabic).
  - Copyright date: prefix "c":  c2005
  - Both dates when differ >= 3 years:  2000, c1996
  - Estimated date: [1980?]
  - Completely unknown: [date unknown]
  - End with period.

--- PAGINATION ---
  Optional; give total pages: 244 p.
  Multiple volumes: 2 vol.
  Unnumbered pages: count and give as  [20 p.]
  No page numbers at all: count leaves: 37 leaves.

--- SERIES ---
  - Capitalize first word + proper nouns only in series name.
  - Series name ; vol. N  or  ; no. N
  - Enclose entire series info in parentheses; period OUTSIDE parentheses.
  - Multiple series: each in own parentheses, separated by semicolons:
    (New directions for evaluation; no. 103); (Jossey-Bass education series).
  - If series has its own editor:
    (Bunting A, editor. Current practice in health sciences librarianship; vol. 8).

--- LANGUAGE ---
  Same rules as journal articles.

STANDARD EXAMPLES:
  Jenkins PF. Making sense of the chest x-ray: a hands-on guide. New York:
  Oxford University Press; 2005. 194 p.

  Eyre HJ, Lange DP, Morris LB. Informed decisions: the complete book of cancer
  diagnosis, treatment, and recovery. 2nd ed. Atlanta: American Cancer Society;
  c2002. 768 p.

  Izzo JL Jr, Black HR, editors. Hypertension primer: the essentials of high
  blood pressure. 3rd ed. Philadelphia: Lippincott Williams & Wilkins; c2003.
  532 p.

================================================================================
PART 4 -- BOOK CHAPTERS / CONTRIBUTIONS (different author from book editor)
================================================================================

FORMAT:
  Chapter author(s). Chapter title. In: Editor(s), editor(s). Book title.
  Edition. Place: Publisher; Year. p. pages.

KEY RULES:
  - Start with the chapter author(s) and chapter title.
  - Use "In:" as the connective phrase (capital I, colon, space).
  - Follow "In:" with full book citation (omit total pagination).
  - Page location: p. 93-113.  (do NOT omit repeated digits across "p."):
      p. 126-7   (NOT p. 126-127)

EXAMPLE:
  Meltzer PS, Kallioniemi A, Trent JM. Chromosome alterations in human solid
  tumors. In: Vogelstein B, Kinzler KW, editors. The genetic basis of human
  cancer. New York: McGraw-Hill; c2002. p. 93-113.

================================================================================
PART 5 -- PARTS OF BOOKS (same author as book; chapters, tables, figures, etc.)
================================================================================

FORMAT:
  Full book citation (omit total pages). Part Name Number, Part title; p. pages.

KEY RULES:
  - Part name: full word, capitalize first letter, NO abbreviation (Figure not Fig.)
  - Part number: Arabic only (VI -> 6).
  - Name + number end with comma + space.
  - Part title: capitalize first word only; end with semicolon + space.
  - If no title: construct from text, enclose in [brackets].
  - If no label/name: use [Figure], [Table], [Photograph] in brackets.
  - Page: p. 438  or  p. XXI-XXII  or  [7 p.]  or  [preceding p. 17]

EXAMPLE:
  Riffenburgh RH. Statistics in medicine. 2nd ed. Amsterdam: Elsevier Academic
  Press; c2006. Chapter 24, Regression and correlation methods; p. 447-86.

================================================================================
PART 6 -- MULTIVOLUME BOOKS
================================================================================

6A. Volume with its own title but NO separate author/editor:
  Full book citation. Vol. N, Volume title. [pages of this volume if needed]

  Tos M. Manual of middle ear surgery. Vol. 3, Surgery of the external auditory
  canal. Stuttgart (Germany): Georg Thieme Verlag; 1997. 305 p.

6B. Volume with its own title AND its own author/editor:
  Treat the set as a series (in parentheses).

  Ollendick T, editor. Children & adolescents: clinical formulation and treatment.
  Amsterdam: Elsevier Science; 1998. 754 p. (Bellack AS, Hersen M, editors.
  Comprehensive clinical psychology; vol. 5).

PART 7 -- CONFERENCE PUBLICATIONS
================================================================================

7A. ENTIRE CONFERENCE PROCEEDINGS

FORMAT:
  Editor(s), editors. Book title. Nth Conference Name; Date; Place.
  City (State): Publisher; Year. Pages.

  -- OR (no separate book title) --

  Editor(s), editors. Nth Conference Name; Date; Place.
  City (State): Publisher; Year. Pages.

CONFERENCE NUMBER:
  Arabic ordinal: Second -> 2nd  III -> 3rd  4e -> 4th  2deg -> 2nd

CONFERENCE TITLE:
  - Capitalize ALL important words (differs from book title rule).
  - Title : Subtitle -> colon + space.
  - Non-English -> [English translation].
  - End with semicolon (if date/place follow) or period (if they don't).

CONFERENCE DATE (optional):
  Format: YYYY Mon DD-DD   e.g.,  2005 Mar 20-24
  Cross-month:  2003 Apr 30-May 2
  Cross-year:   2004 Dec 30-2005 Jan 2
  End with semicolon (if place follows) or period (if no place).

CONFERENCE PLACE (optional):
  City only; US/Canada add state code; international add country name.
  Can optionally prepend hosting institution name.
  End with period.

EXAMPLE:
  van Pelt J, Kamermans M, editors. Development, dynamics and pathology of
  neuronal networks: from molecules to functional circuits. Proceedings of the
  23rd International Summer School of Brain Research; 2003 Aug 25-29; Royal
  Netherlands Academy of Arts and Sciences, Amsterdam, the Netherlands.
  Amsterdam: Elsevier; 2005. 385 p. (Progress in brain research; vol. 147).

7B. PAPER PRESENTED AT A CONFERENCE (published in proceedings)

FORMAT:
  Author(s). Paper title. In: Editor(s), editors. Book title (or conf name).
  Conf number Conf name; Date; Place. City: Publisher; Year. p. pages.

  Abstract:  add [abstract] after paper title.
  Discussion appended:  p. 145-54; discussion p. 155-6.

EXAMPLE:
  Arendt T. Alzheimer's disease as a disorder of dynamic brain self-organization.
  In: van Pelt J, Kamermans M, editors. Development, dynamics and pathology of
  neuronal networks. Proceedings of the 23rd International Summer School of Brain
  Research; 2003 Aug 25-29; Amsterdam. Amsterdam: Elsevier; 2005. p. 355-78.

7C. PARTS OF CONFERENCE PAPERS (tables, figures within a paper)

FORMAT:
  Full paper citation. Part Name Number, Part title; p. pages.
  (Same rules as Parts of Books -- see Part 5 above)

================================================================================
PART 8 -- SCIENTIFIC AND TECHNICAL REPORTS
================================================================================

FORMAT:
  Author(s) (Affiliation). Title. Edition/Report type. Place: Publisher; Year.
  Report No. Contract No. (optional)

KEY DIFFERENCES FROM BOOKS:
  - Author affiliation is often REQUIRED (performing org != publishing org).
  - "Annual Report", "Final Report", "Interim Report" = edition statements.
  - NTIS and GPO are DISTRIBUTORS, not publishers.
    -> Real publisher in Publisher field.
    -> Distribution info in Notes: Available from: NTIS, Springfield, VA; [No.].
  - Report number follows year (after period).

MULTI-INSTITUTION AUTHORS:
  Thompson LA, Chhikara RS (School of Natural Sciences, Univ of Houston,
  Clear Lake, TX); Conklin J (Lyndon B. Johnson Space Center, Houston, TX).

EDITION WITH DATE RANGE:
  Final report 15 Mar 2004-31 Jan 2005.
  Annual summary 1 Jul 2000-31 Dec 2004.

EXAMPLE:
  (standard report with NTIS distribution)
  Acevedo SE, DeVincenzi DL, Chang S, editors. 6th Symposium on Chemical
  Evolution; 1997 Nov 17-20; Moffett Field, CA. Moffett Field (CA): National
  Aeronautics and Space Administration (US), Ames Research Center; [1998]. 105 p.
  (NASA conference publication; 10156).
  Available from: NTIS, Springfield, VA; N19980236014.

PART 9 -- DISSERTATIONS AND THESES
================================================================================

FORMAT:
  Author. Title [dissertation]. [Place]: University; Year. Pages.
  Author. Title [master's thesis]. [Place]: University; Year. Pages.

CONTENT TYPE (required -- placed after title):
  Doctoral:  [dissertation]
  Master's:  [master's thesis]
  On microfiche: [dissertation on microfiche]
             [master's thesis on microfilm]
             [master's thesis on microcard]
  Non-English title with translation: Original title [Translation] [dissertation].

PLACE OF PUBLICATION:
  = City where the degree-granting institution is located.
  Often not printed on thesis -> find via other sources (e.g., Yahoo Education)
  -> place in [brackets]: [Pittsburgh (PA)]:
  Unknown: [place unknown]:

PUBLISHER:
  = Degree-granting institution (may abbreviate Univ. for University).

DATE:
  = Year degree was conferred; may add month: 2000 May  or  2000 Spring.

PAGES:
  Total pages: 290 p.
  Multiple volumes: 2 vol.
  Unnumbered: count leaves: 37 leaves.

NOTES (selected):
  Specific degree:  Master of Public Health.
  Funding:         Sponsored by the Agency for Health Care Policy and Research.
  Availability:    Available from: UMI, Ann Arbor, MI; AAT 3145345.
  Location:        Located at: NLM, Bethesda, MD; W1 AC955 v.20 2001.

EXAMPLES:
  Jones DL. The role of physical activity on the need for revision total knee
  arthroplasty in individuals with osteoarthritis of the knee [dissertation].
  [Pittsburgh (PA)]: University of Pittsburgh; 2001. 436 p.

  Roguskie JM. The role of Pseudomonas aeruginosa 1244 pilin glycan in virulence
  [master's thesis]. [Pittsburgh (PA)]: Duquesne University; 2005. 111 p.

  Montes Alvarez MJ. Parametros predictivos... [Predictive parameters...]
  [dissertation]. Cadiz (Spain): Universidad de Cadiz; 2005. 180 p. Spanish.

PART 10 -- PATENTS
================================================================================

FORMAT:
  Inventor(s), inventor(s); Assignee, assignee. Title. Country patent
  Country-Code Number. Date.

KEY RULES:
  - Inventor before assignee; separated by semicolon.
  - "inventor" / "inventors" after last inventor name.
  - "assignee" (same form singular/plural) after last assignee name.
  - Patent country: adjective form (French, German, Russian).
    Exception: United States  /  United Kingdom.
  - Regional patent offices:
      European Patent Office (EPO)   -> European patent (EP)
      World Intellectual Prop. Org.  -> World patent (WO)
      Eurasian Patent Organization   -> Eurasian patent (EA)
      ARIPO                          -> African Regional Intellectual
                                       Property patent (AP)
  - Patent number: transcribe exactly (preserve spacing/punctuation).
    Letters after number are significant: retain (e.g., EP 0673 422 B1).
  - Date format: YYYY Mon DD   (convert any foreign date format).
  - Patent application: insert "application" between country and code:
    UK patent application GB 2 146 419A. 1984 Sep 4.

OPTIONAL ELEMENTS:
  International classification:  Int. Cl. A61F-002/24.
  National classification:       US Cl. 248/104; 248/102.
  Application number and date:   Appl. No.: 437419; 2003 May 13.
  Related patents:               Related US patents: 6,350,861; 5,714,350.

NON-ENGLISH "INVENTOR" / "ASSIGNEE" EQUIVALENTS:
  French:   inventeur / demandeur, titulaire
  German:   Erfinder / Anmelder
  Italian:  inventore / richiedente
  Russian:  izobretatel / zayavitel
  Spanish:  inventor / solicitante

EXAMPLES:
  Cho ST, inventor; Hospira, Inc., assignee. Microneedles for minimally invasive
  drug delivery. United States patent US 6,980,855. 2005 Dec 27.

  Cameron B, Crouzet J, inventors; Rhone-Poulenc Biochemie, assignee. Cellules
  modifiees au niveau du catabolisme de la betaine... [Cells with altered betaine
  catabolism...]. European patent EP 0 673 422 B1. 2006 Jan 4. French.

================================================================================
PART 11 -- NEWSPAPER ARTICLES
================================================================================

FORMAT:
  Author. Article title. Newspaper Name (Edition). Year Mon Day;Sect. X:Page(col. N).

FIVE KEY DIFFERENCES FROM JOURNAL ARTICLES:
  1. Newspaper name is NEVER abbreviated.
  2. Place of publication must be added if not in the name.
  3. Section replaces volume/issue.
  4. Give starting page only (no end page).
  5. Column number is required: (col. 1)

NEWSPAPER TITLE RULES:
  - Omit leading "The" (optional).
  - Add place if not in name:
      Place inside name:  (Minneapolis) Star-Tribune
                          The (Baltimore) Sun
      Place after name:   Daily Nation (Nairobi)
                          El Dia (Mexico City)
  - US/Canada small city: add state/province: The Record (Stockton, CA)
  - International: add country: Le Journal du Jura (Bienne, CH)
  - National/international papers need no place: Christian Science Monitor

EDITION (required):
  In parentheses after newspaper name; abbreviate "edition" -> Ed.:
    Washington Post (Final Ed.)    Chicago Tribune (Metropolitan Ed.)
  Non-English edition abbreviations:
    French: Ed.  German: Ausg.  Russian: Izd.

DATE:
  Format: YYYY Mon DD; followed by semicolon (if section follows) or colon.

SECTION:
  Sect. A:     Sect. 1:     Sect. C:
  Named section (no letter/number):  Montgomery Extra:   Home & Family:
  No section: date followed directly by colon and page.

COLUMN NUMBER (required):
  (col. 1)  placed after page number, period outside parenthesis.

ONLINE NEWSPAPER:
  Newspaper Name [Internet]. Date [cited YYYY Mon DD];
  Section:[about N p.]. Available from: URL

EXAMPLES:
  Gaul G. When geography influences treatment options. Washington Post
  (Maryland Ed.). 2005 Jul 24;Sect. A:12(col. 1).

  Major decline in U.S. deaths is recorded. New York Times (Washington Final).
  2006 Apr 20;Sect. A:14(col. 6).

================================================================================
PART 12 -- BIBLIOGRAPHIES
================================================================================

FORMAT: Same as Books, with THREE exceptions:

  1. Author role = compiler(s) (not author):
     Last name followed by ,  compiler.  or  ,  compilers.

  2. If "bibliography" is NOT in the title, add content type in brackets:
     [bibliography] after title.

  3. Notes may describe: number of citations, date coverage, source databases.

  French: compilateur  German: zusammensteller
  Italian: compilatore  Spanish: compilador  Russian: sostavitel

PART 13 -- FORTHCOMING / "IN PRESS" MATERIALS
================================================================================

CORE RULES:
  - ONLY cite material that has been ACCEPTED for publication.
  - Do NOT cite material merely submitted but not yet accepted.
  - Notes field MUST begin with the word "Forthcoming".
  - If expected year known:  Forthcoming 2006.
  - If month known:          Forthcoming 2006 May 2.
  - If year unknown:         Forthcoming.

  - Some publishers do not accept forthcoming citations; check target journal.
  - Author affiliation recommended (aids retrieval if publication delayed).
  - If author manuscript is available online, add to Notes:
    Author's manuscript available at http://...

NAME-YEAR SYSTEM OPTION (non-NLM-standard but acceptable):
  Move "Forthcoming YYYY" to immediately after author name.

JOURNAL ARTICLE FORMAT:
  Author(s). Title. Journal Abbr. Forthcoming YYYY.

  Forthcoming 2006.

PART 18 -- INTERNET / ONLINE RESOURCES -- GENERAL RULES
================================================================================

ALL INTERNET RESOURCES -- MANDATORY REQUIREMENTS:

1. MEDIUM TYPE:  Add [Internet] after the title of the host publication.

2. THREE DATES (where applicable):
   a) Date of publication  (year required; month optional)
   b) Date of update/revision -- use exact wording from the document:
      updated / modified / revised / reviewed / amended
      Format:  [updated 2006 May; cited 2007 Mar 5]
      Multiple updates: list chronologically, semicolons between.
   c) Date of citation (REQUIRED for all internet sources):
      [cited YYYY Mon DD]
      Dates b and c share ONE pair of brackets.

3. URL -- "Available from:" format:
   Available from: http://www.example.org/page.html
   - URL ends in slash  -> add period after slash.
   - URL does not end in slash -> no trailing period.
   - Line breaks: break at slash only; NEVER insert hyphen.
   - Multiple URLs: space + semicolon + space between them.
   - Not directly accessible: give navigation instructions:
       Available from: http://... by selecting the Publications link.

4. PAGINATION when not available:
   [about 5 screens]   [about 3 p.]   [6 p.]   [10 paragraphs]
   [174 KB]   [about 3 min.]   (for video)

5. DOI notation (in Notes or after pages):
   doi: 10.1093/jncimonographs/lgi036.
   Working URL form: http://dx.doi.org/10.xxxx/xxxx

PART 29 -- APPENDIX: NLM VS. MEDLINE/PUBMED EXCEPTIONS (10 key differences)
================================================================================

These are cases where PubMed display DIFFERS from Citing Medicine rules:

1. AUTHOR:
   - PubMed (2002+): shows full given names when available.
   - MEDLINE does NOT add ISO country codes to corporate author names.
   - MEDLINE does NOT translate non-English corporate names.
   - No "editor", "translator" role labels in MEDLINE author field.

2. AUTHOR AFFILIATION:
   - 1988-2013: first author only, verbatim, includes "USA" suffix.
   - 2013 Dec+: all author affiliations if publisher supplies them.
   - MEDLINE does NOT standardize; does NOT add city/state/country.

3. ARTICLE TITLE:
   - English-only display (no original language title shown).
   - 2015 Jun 15+: capitalization follows PUBLISHER original style
     (no longer forced to NLM sentence-case rule).
   - Pre-2015: NLM edited 1966+ titles to sentence case.

4. ARTICLE TYPE:
   - MEDLINE does NOT separately index conference abstracts or book reviews.
   - Interview (2015+): interviewee = 1st author; interviewer = 2nd author.

5. JOURNAL TITLE ABBREVIATION:
   - Once established, NLM abbreviations remain even if rules change.
   - Result: some contradictory abbreviations coexist across time periods.

6. EDITION (journal variant):
   - Same as above; older edition notations kept as established.

7. TYPE OF MEDIUM:
   - PubMed DOES NOT display medium type in standard citation view.
   - Viewable in PubMed XML: values are "Print" or "Internet".
   - NLM does not index microfilm, CD-ROM, DVD, or audiovisual journals.

8. PHYSICAL DESCRIPTION:
   - NLM does not collect physical description for electronic formats.

9. LANGUAGE:
   - PubMed citations display in English only; all publication languages noted.
   - Non-English titles shown in [brackets] as translation.

10. LOCATION / PAGINATION:
    - PubMed allows: traditional page range, electronic article number, or both.
    - Examples of article-number formats:
        e11                   (article number as "page")
        pii: 18863            (pii identifier in notes)
        doi: 10.1136/bmj.a295 (DOI as locator)
    - When publisher provides only electronic number, no page range shown.
