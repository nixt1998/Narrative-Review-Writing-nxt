# Research Basis

This skill is grounded in 13 peer-reviewed publications on how to write scientific review articles. The methodology integrates best practices from narrative review guides with selected systematic review methodology where it adds rigor.

## Key Sources

1. Dhillon P. How to write a good scientific review article. FEBS J. 2022;289(13):3592-3602. DOI: 10.1111/febs.16565.
   - Scope checklist, core components table, figure design principles, search/reference tool comparisons

2. Chaney MA. So you want to write a narrative review article? J Cardiothorac Vasc Anesth. 2021;35(10):3045-3049. DOI: 10.1053/j.jvca.2021.06.017.
   - Three quality standards for narrative reviews, NR vs SR distinction, retrograde search method, chronological structure

3. Pautasso M. Ten simple rules for writing a literature review. PLoS Comput Biol. 2013;9(7):e1003149. DOI: 10.1371/journal.pcbi.1003149.
   - Notes-while-reading method, mini vs full review, critical/consistent tone, feedback incorporation

4. Gasparyan AY, Ayvazyan L, Blackmore H, Kitas GD. Writing a narrative biomedical review: considerations for authors, peer reviewers, and editors. Rheumatol Int. 2011;31(11):1409-1417. DOI: 10.1007/s00296-011-1999-3.
   - Narrative biomedical review specifically: authorship criteria, ethics, title/abstract writing, peer review process

## How These Sources Were Used

- **Dhillon (2022)** and **Chaney (2021)** form the primary framework: scope definition, structure, writing quality standards specific to narrative reviews
- **Pautasso (2013)** and **Gasparyan (2011)** contribute process guidance: note-taking, critical evaluation, authorship ethics
- **Goodfellow (2023)** and **Paez (2017)** inform search strategy and database selection
- Systematic review sources (Wright, Harris, Page, Pati, Randles, Livschitz) contribute structured methodology adapted to the narrative review context: PICO thinking, reproducible search, quality assessment logic
- **Balon (2022)** provides the foundational definition of what a review article is and should accomplish

## Reference Format Authority

The citation format used in this skill (Vancouver/NLM style) is based on:

14. Patrias K. Citing Medicine: The NLM Style Guide for Authors, Editors, and Publishers.
    2nd ed. Bethesda (MD): National Library of Medicine (US); 2007. Updated 2020.
    Available from: https://www.ncbi.nlm.nih.gov/books/NBK7256/
    - Authoritative rules for all 30 citation document types (journal articles, books,
      book chapters, theses, patents, preprints, internet sources, and more)
    - Author name rules, punctuation standards, supplement notation, date formats
    - Fully incorporated into the internal vancouver-guide.md skill file (merged 2026-07-09)
    - The internal skill file is the operational reference; this source is the authority
      for any entry type or edge case not covered there

**Usage in skill:** Referenced in Step 1.4 (Reference Format Lock) and Step 9.1 (Reference
Assembly). Not a research paper -- listed separately as a style authority rather than a
methodological source.
