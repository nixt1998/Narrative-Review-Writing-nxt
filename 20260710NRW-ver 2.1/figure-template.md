# Figure Instruction Template (v2.1)

Use this template for each figure in Step 7.0.
Each figure produces three outputs: drawing instructions (06_figure_instructions.md),
plain-text legend (06_figure_legends.txt), and a standalone Word legend file (06_figure_legends.docx).

## Section B: Figure Legend (for 06_figure_legends.txt and 06_figure_legends.docx)

The legend must follow this exact format:

Figure N. [Conclusion statement -- bold in Word]
(A) Description of panel A. (B) Description of panel B. (C) Description of panel C.
Abbreviations: AB, full term; CD, full term; EF, full term.
[If p-values are shown] ns, not significant; *, p < 0.05; **, p < 0.01; ***, p < 0.001.

### Legend Title Rules (critical)

The title MUST be a conclusion statement, not a description of what is shown.

Correct example:
  NLRP3 Inflammasome Activation Mediates Cisplatin-Induced Renal Tubular Cell Pyroptosis

Incorrect example:
  Schematic diagram of NLRP3 inflammasome pathway

### Panel Description Format

Each panel description follows this pattern:
  (A) text ending with period; (B) text ending with period; (C) text ending with period.

### Abbreviation Line

The abbreviation line is MANDATORY for every figure.
List all abbreviated terms used within this figure:
  Abbreviations: NLRP3, NLR family pyrin domain-containing 3; ROS, reactive oxygen species.

If the figure has no abbreviations, write:
  Abbreviations: none.

### Statistical Significance Line

The p-value legend line is MANDATORY if any p-values or significance markers are displayed.
Use this exact wording:
  ns, not significant; *, p < 0.05; **, p < 0.01; ***, p < 0.001.

Note: p is italic lowercase in the Word version.

## Quality Principles (Dhillon 2022)

- One main message per figure
- Labels and scale bars as needed
- Leave adequate white space; do not cram elements
- Consistent font within and across all figures
- Color-blind-safe palette; interpretable in grayscale
